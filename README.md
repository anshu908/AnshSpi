## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/anshu908/Anshsip.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Anshsip
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
